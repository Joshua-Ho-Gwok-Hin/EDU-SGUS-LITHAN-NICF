Admin guide:

  • To creates a NEW USER,
    1. Fill in the form and
    2. Click 'Add New User'
  • To EDIT/MAKE CHANGES to an exisitng user,
    1. Key in User Name
    2. Click "Retrieve Data",
    3. Edit the relavant fields and then
    4. Click "Save"
  • To DELETE an exisitng user
    1. Key in User Name
    2. click "Retrieve Data" (This is to check the correct USER)
    3. Click "Delete"
  • To make *REMOVE ALL USERS
    1. click "Remove all records"







*Warning! This button removes all records!!!
